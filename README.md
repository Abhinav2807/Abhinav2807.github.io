# abhinav2807.github.io

Personal portfolio site for **Abhinav Ayush** — Sr. Founding Engineer & Full Stack Developer.

**Live:** [abhinav2807.github.io](https://abhinav2807.github.io)

## Stack

- Vanilla HTML, CSS, JavaScript (no frameworks)
- Minecraft/Win95 pixel aesthetic with CRT scanline overlays
- VT323 + Space Grotesk + IBM Plex Sans typography
- Canvas-based falling blocks animation
- Bottom hotbar navigation with keyboard shortcuts (1-6)
- Contact form via [Web3Forms](https://web3forms.com)
- SEO meta tags, Open Graph, JSON-LD structured data, llms.txt

## Sections

- **Hero** — Win95 window with player stats and status indicator
- **Experience** — Quest log with dashed-border job cards and tech chips
- **Skills** — RPG inventory slots with XP level bars
- **Education** — Achievement unlock cards with score modules
- **Contact** — Multiplayer chat window with server info sidebar
